# Tetris N-BLOX Offline v1.0.0

First WACZ-based offline release.

## Artifacts

- `tetris-nblox-wacz-offline-1.0.0-mac-x64.zip`
- `tetris-nblox-wacz-offline-1.0.0-win-x64.zip`

## Package Managers

macOS with Homebrew:

```bash
brew tap Svanny/n-blox-offline https://github.com/Svanny/n-blox-offline
brew install --cask tetris-nblox-offline
```

Windows with Scoop:

```powershell
scoop bucket add n-blox-offline https://github.com/Svanny/n-blox-offline
scoop install tetris-nblox-offline
```

Windows with a local WinGet manifest:

```powershell
winget install --manifest .\manifests\winget\Svanny.TetrisNBloxOffline.yaml
```

## Changes

- Rebuilt the offline app from `n-blox-free-tetris.wacz`.
- Runs the captured HTML5/Cocos Tetris N-BLOX game locally through Electron.
- Removes captured ad and analytics loaders.
- Blocks renderer requests outside the local `127.0.0.1` server.
- Adds repeatable local packaging for macOS and Windows x64 without GitHub Actions.
- Adds Homebrew, Scoop, and local WinGet installer manifests.
