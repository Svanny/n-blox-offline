# Notice

This repository packages a local offline wrapper for the Tetris N-BLOX web game assets captured from `www.freetetris.org`.

The wrapper, build scripts, and installer manifests in this repository are maintained here for personal offline use. Third-party game assets, names, trademarks, and audio remain the property of their respective owners.

The app is ad-hoc signed on macOS and unsigned on Windows. Package-manager installs are checksum-backed, but macOS Gatekeeper and Windows SmartScreen may still warn on first launch.
