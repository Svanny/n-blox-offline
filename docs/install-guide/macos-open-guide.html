macOS offline app guide

Only use the official GitHub release zip. Do not disable Gatekeeper globally.

1

Download and unzip

Get tetris-nblox-wacz-offline-1.0.0-mac-x64.zip from the GitHub release.

2

Open from Finder

Right-click Tetris N-BLOX Offline.app, then choose Open.

3

Confirm the prompt

Click Open again only after checking the file came from the official release.

If macOS still blocks it

Open System Settings > Privacy & Security, then use Open Anyway. Apple shows that option for about an hour after the first blocked launch.

Open Anyway

Before opening

Check the SHA-256 checksum in the README. Delete the file if the checksum does not match. Do not run copies from mirrors or unknown websites.
