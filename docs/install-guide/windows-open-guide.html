Windows offline app guide

SmartScreen may warn because this free build is unsigned and has no Microsoft reputation yet.

1

Download and extract

Get tetris-nblox-wacz-offline-1.0.0-win-x64.zip from the GitHub release.

2

Open the app

Open Tetris N-BLOX Offline.exe from the extracted folder.

3

Read SmartScreen

If SmartScreen appears, click More info only after checking the source and checksum.

Only continue if you trust the file

Click Run anyway only if the file came from the official GitHub release and the README checksum matches.

Run anyway

Before opening

Delete the file if the checksum does not match. Do not run modified copies from reposts, mirrors, or unknown websites.
